## How to use?

### 拉取
假设需要拉取gcr.io/google_containers/pause:3.0
```
$ curl -s https://www.zhangguanzhang.com/pull | bash -s -- gcr.io/google_containers/pause:3.0
```
### 查询
#### 查询域名仓库下的namespace和namespace里的镜像列表

}
